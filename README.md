# Escreva seus valores por extenso

# Instalando
``` powershell
Install-Package EscritaPorExtenso 
```

# Importando a biblioteca
``` csharp
using EscritaPorExtenso;
```

# Usando
``` csharp
1.99.PorExtenso(); // "um real e noventa e nove centavos"
```

Simples assim!
